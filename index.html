⚙️ Configurações de Impressão

Epson 80mm — USB / Bluetooth / rede Impressora térmica ligada por USB, Bluetooth ou rede Wi-Fi. Papel 80mm, margem padrão.
Epson RS232 — Porta Série (COM) Impressora térmica ligada por cabo série RS232 (COM1/COM2). Compatível com Epson TM-T20, TM-T88, TM-U220 e similares. Pode requerer configuração de porta no Windows.
ℹ️ Nota RS232: Para impressoras série, certifique-se que:
• A porta COM está configurada no Windows (ex: COM1, 9600bps, 8N1)
• O driver Epson OPOS ou APD está instalado
• O Chrome/Edge tem permissão de acesso à porta série (WebSerial API) se necessário
• Para impressão via browser, o Windows partilha a impressora como impressora de sistema.
